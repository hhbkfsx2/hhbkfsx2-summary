\subsection{Objektorientierung}

Ein Objekt ist eine Instanz einer Klasse. Ein Objekt hat Attribute und beherrscht Methoden. Attribute sind Konstanten oder Variablen mit einem festen Datentyp. In der Regel sind Attribute private, dass bedeutet, dass sie von außen nicht verändert werden können. Daher gibt es zum Ändern von Attributen meist public Methoden, die entsprechende Abfragen enthalten, damit Attribute nur korrekte Daten erhalten. Methoden sind entweder Funktionen oder Prozeduren. Im Gegensatz zu Funktionen liefern Prozeduren keinen Zückgabewert.
